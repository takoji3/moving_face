MovingFace
---

MovingFace is a tool for fun. 
You can play with a face (specify string).

## environment

* python >= 3.*

## Key map

| Key | Descritpin |
|:---:|:---:|
| k,Up | move to top |
| j,Down | move to bottom |
| l,Right | move to right |
| h,Left | move to left |

## Installation

 ```
 $ python setup.py install
 ```
